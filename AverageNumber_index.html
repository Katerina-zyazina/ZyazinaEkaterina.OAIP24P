Нахождение среднего числа

Даны числа: a = {{ a }}, b = {{ b }}, c = {{ c }}

Среднее число: {% if (a >= b and a <= c) or (a <= b and a >= c) %} {{ a }} {% elif (b >= a and b <= c) or (b <= a and b >= c) %} {{ b }} {% else %} {{ c }} {% endif %}
